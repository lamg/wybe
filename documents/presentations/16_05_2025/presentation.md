## What is formal verification?

- Mathematical formulas are universally true

```fs
2 + 2 = 4
```

---

## Programs can be mathematical formulas

```fs
  if ¬true then A else B
↦ { applying ¬ }
  if false then A else B
↦ { semantics of the if statement }
  B
```
